UCI ML Repository Conversion

Besides the ontology and associated services we want to collaborating converting data from public machine learning repositories, such as UCI ML Repository. They currently maintain 313 data sets as a service to the machine learning community. We want to extend the dataid project which aims to describing LOD datasets via RDF files, to host and deliver these metadata files together with the dataset in a uniform way.

For a wide range of experiments

For Natural Language Processing, Stock Market Prediction, Semantic Web Algorithms...MEX defines a set of relations and entities (reusing core W3C Ontologies) in order to describe executions of machine learning algorithms, regardless implementation or technology.

An Interchange Format for Machine Learning

The MEX vocabulary provides a flexible and interoperable format for exporting machine learning output. The metadata information makes the results easier to be understood!
